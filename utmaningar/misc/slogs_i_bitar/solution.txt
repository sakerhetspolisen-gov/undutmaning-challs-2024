## Hints

1. View in CyberChef with:
    - View Bit Plane on bit `0`.
    - Render image.

## Solution

1. Filter *out* pixel values where `LSB != 0`.
2. Reverse each byte (8-bit grayscale encoded pixel).
3. Print bytes as ASCII.
4. (Optional) Read until first `THISISJUSTPADDINGPLEASEIGNORE`.
